# References to Add or Verify Before Submission

The English v8 manuscript should verify and add references for the following model families before journal submission:

- PRnet and its original chemical perturbation prediction task.
- TranSiGen and its basal-expression plus molecular-structure input formulation.
- CPA and chemCPA.
- scGPT.
- scFoundation.
- Geneformer, if included as a frozen embedding arm.
- OpenProblems NeurIPS 2023 single-cell perturbation prediction task.
- Sci-Plex 3 / Srivatsan and Trapnell single-cell chemical perturbation dataset.
- Bemis-Murcko scaffolds and Morgan/ECFP fingerprints.

Do not cite these methods in the final manuscript until each citation has been checked against the exact claim being made.
