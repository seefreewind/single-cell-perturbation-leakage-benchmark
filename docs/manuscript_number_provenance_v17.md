## Known gaps

- Sci-Plex 3 DE-gene-only neural metrics are not reported because the current pseudobulk artifact does not contain per-record DE masks aligned to the 2,000-gene response vectors.
- Training-loss and validation-loss curves were not persisted for the completed neural runs. The current audit can verify train-only validation by code and configuration, but cannot plot historical loss curves without rerunning.
- The Morgan fingerprint bit length is recorded as 1,024 in the local fingerprint matrix. The fingerprint radius is not recorded in the current output metadata and should be added to future preprocessing metadata.
