# PRnet Main10 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade `prnet_adapted_sciplex3` from a seed-0 smoke test to a 10-seed Sci-Plex 3 repeated-seed benchmark using real manifest seeds 1-10.

**Architecture:** Reuse the existing PRnet adapter that imports the official PRnet PGM module, but create a separate main10 runner with real manifest seeds, train-only validation, per-combination failure capture, prediction NPZs, leakage audit, contrast summaries, model-panel integration, and tests. Keep manuscript updates gated until main10 success and QC/source-data criteria are met.

**Tech Stack:** Python 3.9 in `envs/transigen_venv`, PyTorch 2.8 CPU, NumPy, pandas, matplotlib, existing Sci-Plex 3 pseudobulk matrices and split manifest.
